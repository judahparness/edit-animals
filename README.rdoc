== README

This app should be pretty easy to run.

<tt>
Step 1: 
download the files to your preferred directory
</tt>

<tt>
Step 2:
open a new terminal window and navigate to the directory which contains the project files and folders
</tt>

<tt>
Step 3:
rails s in the directory you just navigated to
</tt>

This app was created with Ruby 2.3 and Rails 4.2.6.
